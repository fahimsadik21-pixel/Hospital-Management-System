# Code of Conduct

This project follows a standard code of conduct to maintain a welcoming, respectful, and inclusive community.

## Our Standards

- Be respectful and professional.
- Use inclusive language.
- Collaborate constructively and avoid personal attacks.
- Accept feedback graciously and engage with reviewers politely.

## Reporting

If you experience or witness unacceptable behavior, please report it via a GitHub issue or contact the repository maintainer.

## Enforcement

Project maintainers may remove comments or contributions that violate this code of conduct. Repeated violations can result in being blocked from the repository.
